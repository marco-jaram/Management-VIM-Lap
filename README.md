## Agregar a tu archvo .vimrc los sources

* source ~/.vim/vim-config/1-settings-init.vim
* source ~/.vim/vim-config/2-plugins-install.vim
* sourse ~/.vim/vim-config/3-keymaps.vim
<br>
## Ejecutar desde vim

* :PluginInstall
* :PluginUpdate
* :PluginClean
